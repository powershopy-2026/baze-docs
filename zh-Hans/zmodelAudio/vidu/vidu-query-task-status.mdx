## 概述

根据任务ID查询 Vidu 音频生成任务的状态和结果。此接口专门用于查询通过 `POST /vidu/v1/audio/text2audio` 或 `POST /vidu/v1/audio/timing2audio` 等接口创建的音频任务。

## 使用方法

### 步骤 1：创建音频任务

首先，使用相应的接口创建音频生成任务（例如：[Vidu 文生音频](/zh-Hans/zmodelAudio/vidu/vidu-text-to-audio) 或 [Vidu 可控文生音效](/zh-Hans/zmodelAudio/vidu/vidu-timing-to-audio)）。响应中将包含一个 `task_id`。

### 步骤 2：查询任务状态

使用返回的 `task_id` 查询任务状态：

```bash
GET /vidu/v1/tasks/{task_id}
```

### 步骤 3：获取结果

当任务状态为 `success` 时，响应中的 `creations` 数组将包含生成的结果。每个creation包含：
- `id`: 生成结果ID
- `url`: 生成的音频URL（有效期24小时）
- `cover_url`: 封面图片URL（有效期24小时）

## 任务状态

任务会经历以下状态：
- **created**: 任务创建成功
- **queueing**: 任务排队中
- **processing**: 任务处理中
- **success**: 生成成功（可获取结果）
- **failed**: 任务失败

## 响应结构

响应使用统一的 `creations` 数组结构：

```json
{
  "state": "success",
  "err_code": "",
  "credits": 4,
  "payload": "",
  "creations": [
    {
      "id": "your_creations_id",
      "url": "your_generated_results_url",
      "cover_url": "your_generated_results_cover_url"
    }
  ]
}
```
