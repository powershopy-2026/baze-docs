这个接口用于下载 `baze平台` 中已经完成的 MiniMax 视频内容。平台不会把上游短时效 `download_url` 直接持久化为稳定结果地址，而是在请求时基于保存的 `file_id` 实时换取最新下载链接，再代理返回视频内容。

## 快速开始

```bash
curl --request GET \
  --url https://baze-api.powerbuyin.top/v1/videos/task_abc123/content \
  --header "Authorization: Bearer <token>" \
  --output minimax-video.mp4
```

## 路径参数

| 参数 | 位置 | 必填 | 说明 |
| --- | --- | --- | --- |
| `task_id` | path | 是 | 已完成的视频任务 ID |

## 常见错误

- 任务不存在
- 任务尚未生成可下载内容
- 使用了错误的 `task_id`
