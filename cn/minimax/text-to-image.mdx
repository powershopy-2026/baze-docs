这个接口是 `baze平台` 的统一文生图入口之一，适用于 MiniMax 文生图能力。平台入口是 `/v1/images/generations`，请求会映射到 MiniMax 上游的 `POST https://api.minimax.io/v1/image_generation`。

## 快速开始

```bash
curl --request POST \
  --url https://baze-api.powerbuyin.top/v1/images/generations \
  --header "Authorization: Bearer <token>" \
  --header "Content-Type: application/json" \
  --data '{
    "model": "image-01",
    "prompt": "A cinematic sunrise over a calm harbor",
    "aspect_ratio": "16:9",
    "response_format": "b64_json",
    "n": 1
  }'
```

## 字段映射与能力边界

| 平台字段 | MiniMax 上游字段 | 必填 | 说明 |
| --- | --- | --- | --- |
| `model` | `model` | 是 | 当前仅支持 `image-01` |
| `prompt` | `prompt` | 是 | 图像生成提示词 |
| `aspect_ratio` | `aspect_ratio` | 否 | 宽高比 |
| `width` | `width` | 否 | 目标宽度 |
| `height` | `height` | 否 | 目标高度 |
| `response_format` | `response_format` | 否 | `b64_json` 会映射为上游 `base64` |
| `seed` | `seed` | 否 | 随机种子 |
| `n` | `n` | 否 | 返回图片数量 |
| `prompt_optimizer` | `prompt_optimizer` | 否 | 是否启用提示词优化 |

当前页面只描述平台已经公开支持的字段，不代表 MiniMax 官方文生图能力的全量字段。

## 常见错误

- 使用了未支持的模型
- 请求中缺少 `prompt`
- 同时传入互斥或无效的尺寸参数组合
