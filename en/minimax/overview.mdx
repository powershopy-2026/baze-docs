MiniMax is one of the upstream providers available on the `baze platform`. You can use the platform's unified APIs for text to image, image to image, text to video, image to video, task lookup, and file download without adapting directly to upstream request differences.

## Base URL and authentication

All requests use Bearer token authentication. The base URL is `https://api.powertokens.ai`.

## Unified access model

The `baze platform` exposes stable public endpoint paths. The gateway maps platform fields to MiniMax upstream fields and limits requests to the capabilities that are currently implemented and supported.

## Field mapping principles

- The platform exposes unified field names and stable public paths first.
- Public image and video fields are mapped to MiniMax upstream fields based on the selected capability.
- If a field is not publicly supported on the platform, do not assume it is available just because the upstream provider supports it.

## Capability scope

- Image generation: text to image, image to image
- Video generation: text to video, image to video, start-end to video, subject-reference to video
- Video tasks: query status and download completed output

## Recommended flow

1. Pick the unified endpoint for the capability you need.
2. Send the request with the platform fields.
3. Poll the status endpoint for video tasks.
4. Download the final output after the task is completed.
